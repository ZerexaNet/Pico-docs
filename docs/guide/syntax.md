# 语法参考

## 关键字中英对照

所有关键字中英文等价，可混用：

| 英文 | 中文 | 用途 |
|------|------|------|
| `fn` / `function` | `函数` / `方法` | 定义函数 |
| `let` / `var` | `定义` / `令` | 声明变量 |
| `struct` | `结构体` | 定义结构体 |
| `if` | `如果` | 条件 |
| `else` | `否则` | 条件分支 |
| `for` | `对于` / `遍历` | 循环 |
| `while` | `当` | 条件循环 |
| `return` | `返回` | 返回值 |
| `yield` | `产出` | 生成器 |
| `import` | `导入` | 导入模块 |
| `true` / `false` | `真` / `假` | 布尔值 |
| `nil` / `null` | `空` | 空值 |
| `match` | `匹配` | 模式匹配 |
| `spawn` | `启动` | 创建线程 |
| `try` / `catch` | `尝试` / `捕获` | 错误处理 |
| `and` / `or` / `not` | `且` / `或` / `非` | 逻辑运算 |

## 标点兼容

中文标点自动等价于 ASCII 标点：

```
函数 加法（甲，乙）：   # 等价于 fn 加法(甲, 乙):
    返回 甲 + 乙
```

## 变量声明

```
let x = 10
定义 x = 10
令 x = 10
var x = 10
```

## 代码块风格（三种等价）

```
# 缩进块
fn 加法(a, b):
    return a + b

# 大括号
fn 加法(a, b) {
    return a + b
}

# 开始/结束
fn 加法(a, b) 开始
    return a + b
结束
```

## 数据类型

```
令 整数  = 42
令 浮点  = 3.14
令 字符串 = "你好"
令 布尔  = true
令 空值  = nil
令 列表  = [1, 2, 3]
令 字典  = {name: "pico", version: 1}
```

## 字符串插值

```
令 名字 = "世界"
令 问候 = f"你好，{名字}！"
打印(问候)   # 你好，世界！
```

## 条件

```
if x > 0:
    打印("正数")
else if x < 0:
    打印("负数")
else:
    打印("零")
```

## 循环

```
# 范围循环
for i in 1..10:
    打印(i)

# 列表迭代
for 项 in [1, 2, 3]:
    打印(项)

# 条件循环
令 n = 0
当 n < 5:
    n = n + 1
```

## 函数

```
fn 问候(名字):
    return f"你好，{名字}"

# 匿名函数
令 平方 = fn(x): return x * x

# 异步函数
async fn 获取(地址):
    令 结果 = await 网络.获取(地址)
    return 结果
```

## 结构体

```
结构体 用户:
    名字: str
    年龄: int

    fn 问候(self):
        return f"我是{self.名字}"

令 u = 用户{名字: "小明", 年龄: 18}
打印(u.问候())
```

## 生成器

```
fn 计数(n):
    令 i = 0
    当 i < n:
        产出 i
        i = i + 1

for x in 计数(5):
    打印(x)
```

## 模式匹配

```
匹配 x:
    1 => 打印("一")
    2 | 3 => 打印("二或三")
    _ => 打印("其他")
```

## 错误处理

```
尝试:
    令 结果 = 危险操作()
捕获 err:
    打印("出错了：" + err)
```

## 多线程

```
令 任务 = 启动 fn():
    return 耗时计算()

令 结果 = 任务.等待()
```
